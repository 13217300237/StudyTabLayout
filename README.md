# GreenTabLayout
仿谷歌原生TabLayout开发的绿色版GreenTabLayout
外观上与谷歌的相差无几，但是支持开发过程中所需的大部分UI要求，这些要求，原本如果用谷歌原生TabLayout，实现起来不是那么方便，
现在使用GreenTabLayout可以轻松实现
